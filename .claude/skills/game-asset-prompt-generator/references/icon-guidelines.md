# Icon Guidelines

Use these rules whenever the request is for an inventory icon, ability icon, status icon, or equipment icon.

## Composition

- Single centered subject.
- Strong silhouette with one dominant shape.
- Generous padding so the asset does not touch the frame.
- No clutter, no secondary props unless they improve recognition.

## Detail strategy

- Prioritize recognition at 64x64 pixels.
- Keep materials readable with limited but decisive highlights.
- Use high contrast between key components.
- Simplify small ornament so it survives downscaling.

## Production constraints

- Transparent background.
- No text, borders, labels, or decorative frames unless requested.
- Suitable for extraction into a professional inventory or hotbar UI.
