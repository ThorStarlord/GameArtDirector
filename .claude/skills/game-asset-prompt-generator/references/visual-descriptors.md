# Visual Descriptors Reference

Reference vocabulary for natural-language prompt generation. These terms describe what can be seen — colors, skin, body, perspective, age — for use in plain-text prompts sent to image models.

## Describing Age and Life Stage

### Infant (0-1)

### Toddler (1-3)

### Young Child (4-7)
young child, small child, preschooler, kindergartener — playful, curious, big-eyed, tousled hair, toothy smile, energetic, innocent, round face

### Older Child (8-12)
older child, grade-schooler, school-aged kid, preteen — freckled, active, gap-toothed, lean-limbed, lively, transitional build

### Teenager (13-17)
teenager, adolescent, high school-aged, teenage girl/boy — moody, gangly, early puberty, developing features, serious gaze, youthful but mature, soft jawline

### Young Adult (18-25)
young adult, college-aged, youthful adult — fresh-faced, smooth-skinned, subtly mature, vibrant, minimal wrinkles, soft features

### Adult (26-39)

### Middle-Aged (40-59)
middle-aged, mature adult — distinguished, subtle wrinkles, salt-and-pepper hair, thoughtful, experienced face, graceful aging, defined lines, mature elegance

### Senior (60-79)

### Elder / Advanced Age (80+)
