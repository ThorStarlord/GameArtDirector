# Examples

Use these examples as pattern anchors. Adapt them to the user's request instead of copying them verbatim.

## Example 1

### Input

poison potion icon

### Output

#### Production Safe

Create a 2D fantasy inventory icon of a poison potion in a small glass vial with a rounded body and narrow neck. Fill it with vivid toxic green liquid and a few suspended particles so the effect reads immediately. Keep the silhouette bold and readable at 64x64 pixels, with the bottle centered and isolated on a transparent background. Use clean highlights, clear separation between glass and liquid, and no text or decorative frame.

#### Enhanced Detail

Create a 2D fantasy game icon showing a compact poison potion in a slightly worn glass bottle with a rounded base, narrow neck, and dark cork stopper. The liquid should glow an eerie yellow-green with faint bubbling particles and a subtle mist inside the glass. Add a thin leather wrap around the neck and restrained specular highlights that make the bottle feel crafted but not ornate. Keep the composition centered with generous padding, a strong silhouette, and crisp contrast for readability at small UI sizes. Transparent background, no text, no extra props.

#### Art Director Version

Design a poison potion inventory icon for a dark fantasy RPG, meant to feel dangerous, alchemical, and instantly legible in a crowded inventory grid. Use a compact bottle silhouette with a bright sickly green core, worn glass edges, and a restrained medieval treatment that suggests the potion was mixed by a field apothecary rather than a noble alchemist. Favor one dominant read first, then a few supporting details like trapped particles and a rough cork seal. Centered composition, transparent background, strong readability at 64x64 pixels, no border, no text.

#### Asset Pipeline Version

Create a 2D fantasy game inventory icon depicting a poison potion contained in a small glass vial with a rounded body and narrow neck. The liquid inside is luminous green with subtle swirling particles suspended within it. The bottle is sealed with a worn cork and wrapped with a thin dark leather band. The silhouette should remain instantly recognizable at small inventory sizes. Use clean shapes, controlled highlights, and strong visual contrast between the potion and the bottle. Center the object in the frame with generous padding. Transparent background. No text, no borders, no additional objects. Designed for integration into a professional RPG inventory system.
