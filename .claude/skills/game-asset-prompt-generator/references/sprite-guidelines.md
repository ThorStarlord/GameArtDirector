# Sprite Guidelines

Use these rules for character sprites, enemies, NPCs, props, and other isolated game-ready 2D assets.

## Form and view

- Keep perspective consistent with likely in-game use.
- Use game-ready proportions rather than realistic anatomical complexity.
- Preserve a clean outline and readable internal shape breakup.

## Gameplay clarity

- Make pose, role, and threat level obvious.
- Ensure costume or material blocks read at gameplay scale.
- Avoid noisy background-like detail that fights the silhouette.

## Production constraints

- Transparent background.
- Single asset focus unless the user requests a sheet or multi-part composition.
- Clean extraction and integration into a 2D pipeline.
